# CivicPulse – Plataforma de Inteligência Urbana

CivicPulse é uma plataforma moderna de inteligência urbana desenhada para visualizar e gerir dados municipais de forma eficiente e intuitiva.

## 🚀 Funcionalidades
- **Dashboard de Monitorização**: Visualização de métricas urbanas em tempo real.
- **Análise de Dados**: Ferramentas avançadas para processamento de informação citadina.
- **Interface Premium**: Design moderno com glassmorphism e foco na experiência do utilizador.

## 🛠️ Tecnologias
- HTML5 Semântico
- CSS3 (Vanilla com Variáveis e Modern Design Patterns)
- JavaScript (Interatividade e Lógica)

## 📦 Como Executar
Basta abrir o ficheiro `index.html` em qualquer navegador moderno. Para uma melhor experiência de desenvolvimento, recomenda-se o uso da extensão "Live Server".
